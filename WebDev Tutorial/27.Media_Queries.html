Media queries are used when we want to customize our website's presentationaccording to the user's screen size. With the help of media queries, you can displaydifferent markups based upon the device's general type(mobile, desktop, tablet).

A media query is a logical operation. Whenever a media query becomes true, then therelated CSS is applied to the target element.
